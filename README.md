# PM-interview-prep

> A living, structured knowledge base for Product Management interview preparation — built and maintained by an AI PM actively looking for new roles.

---

## 👤 About This Repo

I'm an early-career PM (0–3 years experience) using this repo to systematically prepare for PM interviews at product-led companies.

If you're also prepping, feel free to fork and adapt this for your own journey.

---

## 🔑 PM Competencies I'm Building

These are the core skills PM interviewers evaluate. Each section of this repo maps back to at least one of these:

- **Product Sense** — Can you design products users love?
- **Strategic Thinking** — Can you see the big picture and prioritize?
- **Analytical Thinking** — Can you define, track, and interpret metrics?
- **Execution** — Can you ship things and handle ambiguity?
- **Communication** — Can you influence without authority?
- **Customer Empathy** — Do you deeply understand user problems?
